# Sprint 0 — PoC (Proof of Concept)

> **기간:** Week 1~2 | **마일스톤:** M0
> **목표:** AI 분석 코어 기술 검증 및 프로젝트 기반 확립
> **Velocity 목표:** 21 SP (+ Karpathy 보완 +6 SP = **27 SP**)

---

## Sprint Goal

> LLM API를 연동하여 텍스트 요구사항 입력 → 구조화된 분석 결과 출력이
> 가능한 프로토타입을 완성하고, 이후 스프린트의 기술적 불확실성을 제거한다.
