# Sprint 3 — 파일 업로드 + 누락 탐지

> **기간:** Week 7~8 | **마일스톤:** M2 (v1.1 1/2)
> **목표:** 문서 파싱(PDF/DOCX/TXT) + 파일 업로드 UI + 누락 가능 요구사항 탐지
> **Velocity 목표:** 22 SP

---

## Sprint Goal

> 사용자가 PDF, DOCX, TXT 파일을 드래그 앤 드롭으로 업로드하면
> 텍스트를 추출하여 AI 분석을 수행하고, 누락 가능 요구사항까지 탐지한다.
