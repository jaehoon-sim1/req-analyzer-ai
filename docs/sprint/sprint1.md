# Sprint 1 — AI 코어 + UI 기반 구축

> **기간:** Week 3~4 | **마일스톤:** M1 (MVP 1/2)
> **목표:** AI 분석 파이프라인과 UI 기본 레이아웃을 구축하여 텍스트 입력 → 결과 표시 흐름 완성
> **Velocity 목표:** 29 SP (+ Karpathy 보완 +5 SP = **34 SP**)

---

## Sprint Goal

> 사용자가 텍스트 요구사항을 입력하고 [분석 시작] 버튼을 누르면,
> AI가 요약·기능 목록을 생성하여 화면에 구조화된 형태로 표시한다.

- **Given** 사용자가 텍스트 요구사항을 입력한다
